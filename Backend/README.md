# Server Module

This directory contains the backend server implementation with an Express app, middleware, modules, jobs, and configuration.

## Structure

- `src/config` - environment and database configuration
- `src/modules` - core business logic by domain
- `src/middleware` - request/response middleware
- `src/jobs` - scheduled CRON jobs
- `src/utils` - shared utilities
- `src/constants` - application constants
- `src/app.js` - Express application entry point
- `src/server.js` - server startup script
